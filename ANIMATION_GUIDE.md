# ANIMATION GUIDE

## Hover
Use subtle hover transitions for buttons, links, and cards.

## Page Transitions
Apply smooth fades and motion that preserve content clarity.

## Loading
Use minimal loading states with simple progress indicators.

## Parallax
Use motion sparingly and only when it adds context or depth.

## GSAP
Use motion libraries for refined animation control where needed.

## Scroll Animations
Animate sections into view with clean, non-distracting motion.
